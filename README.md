# Currency-Converter-using-JAVA
A Currency Converter using advanced JAVA. we'd be using AWT and swing components for a smooth GUI.

A Currency Converter is a semi-advanced JAVA program to show that you have grasped the contents of JAVA. 
It is one of most common and obvious use of programming to solve a real-life problem.

This projects implements textbox, a dropdown menu ( ComboBox), buttons, LAbels and JTabbed Pane.

THe project consists of 2 modules:
1. A currency converter that converts currencies Ruppe, Pounds, USD, Euros, Yen and Won to each other.
2. A JDBC Module that lets you search a currency of the given country

Make sure to connect the project to your MySQL to produce results
